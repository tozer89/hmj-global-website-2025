Upload guidance

Improve match quality

The strongest results come from concise, relevant evidence rather than volume. Use this area to give the matcher the clearest picture of the candidate.

  • Upload the latest CV or resume as the core candidate document.
  • Add certificates, qualifications, or trade documentation when they are role-critical.
  • Certification photos or qualification screenshots can be included as supporting image evidence.
  • Include a cover letter only if it adds useful context on mobility, rates, or sector interest.
  • Use recruiter notes for constraints, concerns, or commercial context the documents do not cover.

    Handling and privacy

    What this module exposes

    Server-side AI only

    OpenAI calls are made from secure server functions. No API key is shipped in admin HTML, browser JavaScript, or public bundles.

    Published roles only

    The matcher compares candidates only against jobs that are both published and currently live in the HMJ jobs source.

    Private storage path

    Saved uploads and results are intended for admin-only review and should remain in private Supabase storage/tables.
